Crop Calculator

This tool will show the cost and benefit of your harvest as well as where to buy them, the number of waters, optimal biome with growing time reduction, and growing time.

How to use:

1. Select the crop.

2. Set the amount.

3. Click "CALCULATE".

4. Click "RESET" to start again.

⚠️Attention: This tool cannot calculate duplicated crops by companions.⚠️
